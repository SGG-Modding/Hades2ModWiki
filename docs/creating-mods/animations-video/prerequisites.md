# Prerequisites

Necessary tools before you can start adding new models, animations or full-screen videos to the game.

:::danger[Current limitations]
At the time of writing, there is no way to export or import 3D models from/into Hades II. The guides in this category are limited to Bink (`.bik`) video files for 2D animations and full-screen videos.
:::

> Parts of this guidance were adapted from [EtchJetty/Girl Bowser's BinkFixTutorial](https://github.com/EtchJetty/BinkFixTutorial) for Hades.

---

## Tools

To work with Bink Video (`.bik`) files, you need `RAD Video Tools` by Unreal, which you can download from [www.radgametools.com/bnkdown.htm](http://www.radgametools.com/bnkdown.htm).
Please note that a password is required to unpack the downloaded archive.
The password is provided on the download page.
